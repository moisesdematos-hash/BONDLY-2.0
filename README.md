# Bondly 2.0 - Premium Edition ✨🚀

Esta é a versão **CONSOLIDADA e COMPLETA** do Bondly, unificando o projeto base com as melhorias premium da versão 2.0.

## 🌟 O que há de novo no 2.0?

- **Interface Glassmorphism de Luxo:** UI redesenhada com transparências, desfoques e gradientes premium.
- **Novas Funcionalidades:** Agreements, Gratitude, Wishlist e o AI Date Planner.
- **Ecossistema Unificado:** Landing Page, Backend (NestJS), Database (SQL) e Frontend (Flutter) em um só lugar.
- **Visão Geral Interativa:** Veja `landing/overview.html` para uma prévia completa das telas.

## 🚀 Estrutura do Projeto

O ecossistema Bondly 2.0 é composto por:

1.  **[Backend (NestJS)](./backend):** API robusta construída com Node.js, TypeScript e integrada com Supabase (PostgreSQL) e OpenAI.
2.  **[Frontend (Flutter)](./frontend):** Aplicação móvel multiplataforma focada numa experiência de utilizador premium e animações fluidas.
3.  **[Landing Page](./landing):** Página web moderna para apresentação do produto e conversão de novos utilizadores.

---

## ✨ Funcionalidades Principais

### 🎮 Gamificação e Interação
-   **The Daily Question:** Perguntas diárias para gerar conversas profundas.
-   **Desafios de Casal:** Atividades propostas para quebrar a rotina e ganhar XP.
-   **O Nosso Jardim:** Um espaço virtual evolutivo que cresce com a atividade do casal.

### 🧠 Inteligência Emocional (IA)
-   **Mood Check-ins:** Registo diário de humor com acompanhamento de tendências.
-   **AI Coach & Insights:** Relatórios mensais detalhados gerados por IA baseados no histórico emocional do casal.
-   **Icebreakers & SOS:** Ferramenta para quebrar o gelo em momentos de tensão no chat.
-   **Simulador de Conversas:** Pratique diálogos difíceis com feedback de um coach virtual.

### 🛠️ Utilidade e Organização
-   **Chat em Tempo Real:** Comunicação direta com análise de sentimento integrada.
-   **Mural de Memórias:** Espaço para guardar fotos, áudios e vídeos de momentos especiais.
-   **Wishlist Partilhada:** Lista de desejos conjunta com links diretos para lojas.
-   **Acordos do Casal:** Definição e aceitação formal de regras e limites da relação.

---

## 🛠️ Tecnologias Utilizadas

-   **Backend:** NestJS, TypeScript, JWT, Supabase SDK, OpenAI API, Stripe (Pagamentos).
-   **Frontend:** Flutter, Riverpod (Estado), Supabase Flutter, fl_chart, url_launcher.
-   **Base de Dados:** PostgreSQL (via Supabase) com Row Level Security (RLS).
